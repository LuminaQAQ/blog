项目结构
----

* `db` 文件夹：数据库文件

* `server` 文件夹：
  安装：  
  
  ```shell
  npm install
  ```
  
  启动： 
  
  ```shell
  npm run serve
  ```

* `vue_admin` 文件夹：
  安装：  
  
  ```shell
  npm install
  ```
  
  启动：  
  
  ```shell
  npm run serve
  ```

* `vue_user` 文件夹：
  安装：  
  
  ```shell
  npm install
  ```
  
  启动：
  
  ```shell
  npm run serve
  ```
